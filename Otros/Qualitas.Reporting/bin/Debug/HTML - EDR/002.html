Welcome to Qualitas Reporting
Choose an option, or browse the menu through the application button
 
Build a new query Manage connections
This option will allow you to build a new query with the Query Designer, by writing
SQL code or following the designer's wizards steps.
 		 Configure different database connections used by queries that are built on
Qualitas Reporting.
Execute a query Manage security
Opens a query from the queries repository in execution enviroment.
Make sure you have enough permissions to execute queries.

 Add, modify, lock or suspend users. You  may also indicate which queries are
allowed to be executed by a user and which are not.
 
Design a query Queries categories
Opens a query in design mode.
This option should be reserved only for the system administrator.
 		 Organize queries according to the categories you consider, by giving
certain logic to the final user.
Remove a query User preferences
This option will remove a query from the repository, together with all its access
permissions and features.

 This option allows you to set current user's preferences, such as the
application theme, language and other options.
Import a query General configuration
From here you can add new queries to the repository from disk files.

 		 Configuration options that affect all users settings. This should only
be reserved for system administrators.
Export a query Product documentation
Unlike the import feature, this option will save a query definition to a disk file, e.g. for
backup purposes or to send it by e-mail.
 		 Access Qualitas Reporting online help and learn all its features to take
full advantage of this product.
