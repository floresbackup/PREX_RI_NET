Bienvenido a Qualitas Reporting
Seleccione una opción, o navegue por el menú a través del botón de la aplicación

  
 
Crear nueva consulta Administrar conexiones
Esta opción le permitirá crear una nueva consulta mediante el diseñador de consultas,
ya sea mediante la escritura de código SQL o bien, de seguir los pasos de un asistente.
 		 Configure las diferentes conexiones a bases de datos que utilizarán las
consultas creadas en Qualitas Reporting.
 
Ejecutar una consulta Administrar la seguridad
Abrir una consulta desde el repositorio en modo ejecución.
Asegúrese contar con los permisos necesarios para ejecutar consultas.
 		 Agregue usuarios y defina los permisos de cada uno. Indique también
cuáles son las consultas que puede ejecutar cada usuario y cuáles no.
 
Diseñar una consulta Categorías de consultas
Abrir una consulta en modo diseño. Esta opción deberá estar
reservada solamente para el administrador del sistema.
 		 Ordene las consultas de acuerdo a las categorías que considere
conveniente, a fin de indicar una lógica al usuario final.
 
Eliminar una consulta Preferencias del usuario
Esta opción quitará una consulta del repositorio, junto con todos sus permisos de
acceso y características
 		 Esta opción permite establecer las preferencias para el usuario actual, tales
como el tema de aplicación a utilizar, idioma y otras opciones.
 
Importar una consulta Configuración general
Desde aquí podrá incorporar nuevas consultas al repositorio a partir de archivos
externos.
 		 Opciones de configuración que afectan a todos los usuarios.
Deberá reservarse solo para el administrador del sistema.
 
Exportar una consulta Documentación del producto
A diferencia de importar, esta opción le permitirá grabar un archivo con toda
la información referida a una consulta.
 		 Acceda a la ayuda en pantalla de Qualitas Reporting y conozca todas
las funciones que le permitirán aprovecharlo al máximo.
