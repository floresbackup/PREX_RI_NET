Las variables, también conocidas como variables globales, constituyen una herramienta para flexibilizar el código SQL de la consulta, permitiendo incluir en el mismo valores que dependerán de ciertos factores, de acuerdo a que variable se trate.

Las variables se pueden utilizar en cualquier ámbito de proceso de una instrucción SQL. Esto quiere decir, que dentro del diseño de una consulta, se podrán utilizar variables en:

  • Instrucción SQL Previa
  • Instruccion SQL Posterior
  • Instruccion SQL Inicial
  • Instruccion SQL Final
  • Instrucción SQL de un parámetro
  • Valores de validación de un parámetro
  • Instrucción SQL de ayuda contextual de un parámetro
  • Valores predeterminados de un parámetro

Todas las variables globales se caracterizan por estar encerradas por el caracter "|" (pipe).
