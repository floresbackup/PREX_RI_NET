Grupo Común

  • Nueva consulta: inicia el diseño de una nueva consulta
  • Abrir: presenta el cuadro de diálogo de selección de consulta, para abrir una de ellas en modo de ejecución.
  • Diseñar: presenta el cuadro de diálogo de selección de consulta, para abrir una de ellas en modo de diseño.
  • Eliminar: presenta el cuadro de diálogo de selección de consulta, a fin de eliminar una consulta del repositorio.

Grupo Importar y exportar

  • Importar: abre la ventana de importación de consultas para agregar una de ellas a partir de un archivo en disco.
  • Exportar: presenta el cuadro de diálogo de selección de consulta, para exportar una de ellas a un archivo en disco.

Grupo Configuración

  • Preferencias del usuario: presenta la ventana de configuración de opciones a nivel de usuario, tales como el idioma, el tema de aplicación y otras.
  • Configuración general: opciones de configuración que afectan a todos los usuarios del sistema.

Grupo Otros

  • Usuarios y seguridad: accede a la administración de usuarios y permisos.
  • Conexiones: ingresa a la ventana de administración de conexiones a bases de datos.
  • Categorías: permite crear, renombrar y eliminar categorías. Asimismo, desde aquí se puede modificar también la categoría de una consulta puntual.

Grupo Ver

  • Menú desplegable que ofrece la posibilidad de mostrar u ocultar los paneles del repositorio de consultas y de consultas favoritas. Ofrece también una lista de las ventanas abiertas para poder intercambiar entre ellas.

Grupo Ayuda

  • Contenido: accede a este documento de ayuda.
  • Acerca de la versión: información específica de la versión, derechos de autor y copyright.

La barra de herramientas se puede ocultar o mostrar haciendo doble clic en la solapa denominada "Consultas". Este comportamiento es aplicable a todas las barras de herramientas en forma de cinta a través de toda la aplicación.
