Las conexiones a bases de datos permiten a Qualitas Reporting obtener los datos definidos en los diseños de consulta y presentarlos al usuario. De esta manera, la herramienta ofrece múltiples opciones para conectar con una base de datos, ya sea Microsoft SQL Server, Oracle, un proveedor OLEDB o una conexión accesible mediante ODBC (Open DataBase Connectivity).

Para agregar una conexión, ingrese al menú Conexiones y presione el botón Agregar:

Una conexión consta de los siguientes atributos:

  • Nombre: es el nombre que identificará una conexión de otra. Este atributo es requerido.
  • Descripción: provee un espacio para describir la conexión, como por ejemplo, "Conexión al servidor de pruebas".
  • Tipo de conexión: especifica a que tipo de motor de base de datos se conectará. Las opciones posibles son:
    • MS SQL Server
    • Oracle
    • OLEDB
    • ODBC
  • Permitir ejecución asíncrona: esta opción indica que la ejecución de la consulta se hará en segundo plano, sin ocupar todos los recursos disponibles del sistema y con la posibilidad de cancelar la misma antes de su finalización. Se encuentra habilitada sólo cuando el tipo de conexión seleccionado es MS SQL Server. Nótese que esta opción sólo surtirá efecto si en la cadena de conexión se especifica un modificador que lo habilite, que por lo general es "Asynchronous Processing=True;".
  • Utilizar opciones predeterminadas: de activarse esta opción, el procesador de consultas de Qualitas Reporting asumirá el símbolo decimal, el literal de cadenas, el literal de fechas y el formato de fechas de acuerdo a lo indicado en la configuración general.
  • Cadena de conexión: representa el "camino" hacia la base de datos. Es aquí donde se indica el nombre del servidor (o su dirección IP), el nombre de la base de datos a utilizar, el tipo de seguridad que se utilizará y otras opciones específicas que dependerán del tipo de conexión utilizado.
    Para configurar correctamente la cadena de conexión, se sugiere seguir los pasos del asistente mediante el botón Generar conexión.
