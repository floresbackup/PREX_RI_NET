Para una adecuada implementación de Qualitas Reporting, se enumeran a continuación los recursos necesarios de acuerdo a las mejores prácticas.

  • DE HARDWARE
    • Para estaciones de trabajo
      • Procesador Intel Pentium IV 1.7 Ghz o superior
      • 1 GB RAM o superior
      • Resolución de pantalla de 800 x 600 a 65536 colores o superior
      • Teclado y Mouse
    • Para el servidor de aplicaciones
      • Procesador Intel Pentium IV 1.7 Ghz o superior
      • 1 GB RAM o superior
  • DE SOFTWARE
    • Para estaciones de trabajo
      • Sistema operativo Microsoft Windows 2000 SP4 o superior
      • Microsoft .NET Framework 2.0
  • HUMANOS
    • Para el diseño de consultas
      • Contar con conocimientos del lenguaje SQL para el desarrollo de consultas complejas, o de nociones elementales de bases de datos para dar curso a los asistentes para la generación de consultas SQL
