Qualitas Reporting enumera varias características, a saber:

  • Está desarrollado bajo una plataforma de vanguardia (.NET Framework) y utiliza componentes de última tecnología.
  • Soporta múltiples idiomas como interfaz de usuario.
  • También soporta acceso a múltiples sistemas de administración de bases de datos, como:
    • Microsoft SQL Server (de forma nativa)
    • Oracle (de forma nativa)
    • Conexiones mediante OLEDB
    • Conexiones mediante ODBC
  • Incorpora el renovado look and feel de Microsoft Office 2007 y sus nuevas barras de herramientas.
  • Posee tanto un esquema de seguridad propia como integrada a Windows mediante Active Directory.
  • Es capaz de manipular grandes cantidades de datos (tablas de hasta más de un millón de registros). 
  • Propicia la creatividad del diseñador mediante el lenguaje SQL, y lo asiste en:
    • Redacción de código mediante Intellisense (ayuda contextual de objetos de la base de datos a medida que se escriben las instrucciones SQL).
    • Diseñador visual (contenedor visual de todos los elementos de la base de datos que escribe automáticamente la instrucción SQL).

Qualitas Reporting otorga mediante la versatilidad de sus componentes -como grillas de datos, tablas dinámicas, gráficos e informes- una solución ideal para la visualización y modelado de datos sin la necesidad de recurrir a complejos y costosos sistemas de Datawarehousing.

Esta herramienta apunta a resolver tanto las necesidades de los usuarios operativos de una empresa, como aquellos de la más alta línea gerencial. A modo de ejemplo, se puede utilizar para objetivos tales como, desde analizar un registro de accesos a un sistema determinado, hasta medir la rentabilidad de un producto en un plazo de tiempo.
